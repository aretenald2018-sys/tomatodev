# 라이프존 연속 기록 전광판 에셋 계획

`docs/ai/NPC_ASSET_WORKFLOW.md`의 공간/소품 체크리스트를 전광판 오버레이에 적용한다.

1. 홈 위치: `lz-scene`의 둥근 카드 내부 좌측 상단이며, 카드 폭의 약 48~50%를 사용한다. 라이프존 1672 기준으로는 화면 좌측 약 0~830px, 상단 약 0~320px 영역에 대응한다.
2. 겹침: 상단 좌측 벽과 창 일부를 의도적으로 덮지만 NPC, 운동 캐릭터, 식탁 캐릭터와는 겹치지 않는다.
3. 홈/모달 분리: NPC 작업이 아니므로 모달용 자산은 만들지 않는다. 홈 배치용 전광판 1종만 만든다.
4. 전용 소품: 빈 LED 화면을 가진 전광판 본체를 RGBA PNG 오버레이로 만든다. 동적 문구와 숫자는 이미지에 굽지 않고 DOM으로 렌더한다.
5. 원근: 베이스룸의 좌우 26~30도 아이소메트릭 축을 따르는 좌향 45도 판면, 검은 금속과 짙은 목재 프레임, 한쪽 지지대와 받침으로 구성한다.
6. 크기: 390px 모바일 카드에서 약 185~195px 너비로 보이게 하며, 카드 좌측 절반을 빈틈 없이 채우는 크기를 기준으로 한다.
7. 이름표/전구: 해당 없음. 연속 기록 문구, 수치, 안내 버튼은 기존 DOM과 이벤트를 유지한다.
8. 정적 자산: 새 PNG를 `assets/home/life-zone/ui/`에 저장하고 `runtime-assets.js`에 등록한다. JS/CSS 변경도 정적 자산에 포함된다.
9. 캐시: PNG, JS, CSS가 바뀌므로 `sw.js`의 `CACHE_VERSION`을 같은 변경에서 올린다.
10. 완료 증거: `http://localhost:3000`의 HTTP 200과 로그인된 홈 화면에서 카드 좌측 절반 배치, 각도, 둥근 모서리 잘림을 확인한다. 운영 배포는 사용자가 명시적으로 요청할 때만 수행한다.
