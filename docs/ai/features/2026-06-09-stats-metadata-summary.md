# 전체통계 메타데이터 요약 계획

## 요청 원문

전체 통계 최상단에 여기 적혀있는 메타데이터들을 그대로 집계해서 이용자들에게 제공하게 해줘

## 이해한 내용

- 목표: `전체통계` 탭 최상단에 스크린샷의 다이어트 상세 리포트 항목들을 현재 토마토팜 기록 데이터에서 집계해 보여준다.
- 비목표: 새 입력 필드 추가, 관리자 통계, 배포, `www/` 직접 수정은 하지 않는다.
- 사용자 흐름: 사용자가 `통계 > 전체통계`를 열면 CSV 버튼과 기존 운동 통계보다 먼저 `전체 기록 리포트`를 본다.
- 데이터 가정: 전체 기록 기준으로 음식/칼로리/운동/평균 영양소를 계산하고, `이달의 감량`은 현재 월의 첫 체크인과 마지막 체크인을 비교한다.
- 열려 있는 질문: 별도 기간 선택 UI가 필요한가? 이번 슬라이스에서는 만들지 않는다.

## 그릴 결과

- 적용 트리거: `/grill-me`
- 핵심 질문: 요약 범위가 월간인지 전체 기록인지가 UI와 계산을 바꾸는가?
- 추천 답변: `전체통계` 최상단이므로 전체 기록 기준으로 집계하고, 스크린샷의 `이달의 감량` 행만 현재 월 기준으로 계산한다.
- 사용자 답변: 명시 답변 없음. 요청의 위치가 `전체 통계 최상단`이므로 추천 답변을 적용한다.
- 확정된 결정: 전체 기록 요약 카드 1개를 추가한다.
- 남은 가정: 저장 키가 없는 `골격근량`, `걸음 수`, `물 섭취량`, `배변 횟수`는 임의 추정하지 않고 저장 데이터가 없으면 `데이터가 존재하지 않습니다.`로 표시한다.

## 결정 기록

- 결정: 기존 `stats-block`/`diet-stat-row` UI를 재사용한다.
- 이유: 통계 화면의 시각 패턴을 유지하고 변경 범위를 작게 유지한다.
- 되돌릴 수 있는가: 가능. `index.html`의 컨테이너와 `render-stats.js`의 렌더 함수 제거로 되돌릴 수 있다.

## 실행 슬라이스

### 슬라이스 1: 전체통계 최상단 메타데이터 요약

- 목표: 스크린샷 항목을 전체통계 최상단 카드로 렌더링한다.
- 범위:
  - 가장 많이 먹은 음식 및 칼로리
  - 가장 많이 먹은 날
  - 운동을 가장 많이 한 날
  - 평균 음식 칼로리
  - 평균 운동 칼로리
  - 평균 몸무게, 골격근량, 체지방량
  - 이달의 감량 몸무게, 골격근량, 체지방량
  - 평균 걸음 수
  - 평균 물 섭취량
  - 평균 배변 횟수
  - 평균 영양소별 섭취량
- 예상 수정 파일:
  - `index.html`
  - `render-stats.js`
  - `sw.js`
  - `docs/ai/NEXT_ACTION.md`
- 수정하지 말 것:
  - `www/`
  - 새 저장 필드/Firestore 구조
  - 기존 운동/식단 저장 로직
- 구현 메모:
  - `getCache()`의 날짜 키만 순회한다.
  - 음식명 빈 값은 제외한다.
  - 운동 칼로리는 `calcBurnedKcal(day, weight)`를 사용한다.
  - 체지방량은 `weight * bodyFatPct / 100`으로 계산한다.
  - 골격근량은 기존 저장 필드가 있으면 읽고, 없으면 데이터 없음으로 표시한다.
  - 당류/나트륨은 저장된 day-level 또는 food item 값을 합산할 수 있을 때만 표시한다.
  - `index.html`과 `render-stats.js`는 `STATIC_ASSETS` 대상이므로 `sw.js` `CACHE_VERSION`을 범프한다.
- 검증 방법:
  - `node.cmd --check render-stats.js`
  - `node.cmd --check sw.js`
  - `npm.cmd test -- --runInBand`가 없으면 기존 테스트 명령을 확인 후 가능한 범위 실행
  - 브라우저 검증은 사용자 로컬 터미널에서 `npm.cmd run dev` 실행 후 `통계 > 전체통계` 확인
- 완료 증거:
  - `전체 기록 리포트` 카드가 전체통계 최상단에 렌더링된다.
  - 저장 데이터가 없는 항목은 데이터 없음 문구로 표시된다.
  - 정적 자산 변경에 맞춰 캐시 버전이 갱신된다.
- 다음 세션 시작 프롬프트:
  - `docs/ai/features/2026-06-09-stats-metadata-summary.md`의 슬라이스 1을 실행한다.
