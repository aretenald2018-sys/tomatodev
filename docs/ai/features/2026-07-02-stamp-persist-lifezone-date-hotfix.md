# 2026-07-02 완료 도장 유지 및 라이프존 날짜 복구 핫픽스

## 문제

- 실제 배포 기준 `tomatofarm/main`에는 이전에 말했던 완료 도장 유지 수정이 없었다.
- 성장 보드 완료 저장은 실패를 호출부까지 전파하지 않아, 도장이 잠깐 보인 뒤 서버 상태 재반영 시 사라질 수 있었다.
- `saveTestBoardV2`가 `test_board_v2` 전체를 덮어써서 오래된 보드 저장이 이미 찍힌 `paintedAt` 로그를 지울 수 있었다.
- 오늘의 라이프존 제목 영역이 배우 이름으로 대체되어 연도 포함 날짜가 사라졌다.

## 범위

- 완료 버튼은 세트 상태를 토글하지 않고 명시적으로 완료 상태로 설정한다.
- 성장 보드 완료 도장은 보드 저장이 성공한 뒤에만 확정한다.
- `test_board_v2` 저장 시 기존 완료 로그와 병합한다.
- 라이프존 헤더를 날짜, 제목, 이름 줄로 분리한다.
- `sw.js` 캐시 버전을 올려 배포 브라우저가 새 파일을 받게 한다.
