# 홈 러닝 GPS 지도 배율 조정 계획

## 배경

- 사용자는 홈탭에서 러닝 기록을 보여주거나 러닝 중일 때의 GPS 지도 말풍선을 첨부 화면 수준의 넓은 배율로 보여 달라고 요청했다.
- 이 배율은 운동 탭 내 러닝 결과 GPS 배율과 별도로 관리해야 한다.

## Slice 1. 홈탭 러닝 지도 전용 배율

1. 홈 라이프존 러닝 지도 말풍선 렌더링 경로를 확인한다.
2. 운동 탭의 `renderRunningMap`/러닝 결과 지도 배율은 변경하지 않는다.
3. 홈탭 지도 말풍선에만 별도 padding/scale 옵션을 적용해 더 넓은 동네 단위가 보이게 한다.
4. 라이브 러닝과 완료된 러닝 기록 모두 같은 홈탭 전용 배율을 사용하게 한다.
5. 회귀 테스트는 홈탭 전용 zoom/padding 계약과 운동 탭 지도 계약 분리를 검증한다.

## 검증

1. `node --check home/life-zone.js`
2. 관련 홈 러닝/러닝 지도 테스트 실행
3. 전체 테스트 또는 영향 범위 테스트 실행
4. `sw.js` `STATIC_ASSETS` 포함 파일 변경 시 `CACHE_VERSION` 갱신
5. Dashboard3 Pages 배포 검증
