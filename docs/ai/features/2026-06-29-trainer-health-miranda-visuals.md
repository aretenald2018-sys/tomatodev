# 트레이너 통계 모달·건강지표 롤백·미란다 모달 아트 개선 계획

## 배경

- 트레이너 통계 모달의 캐릭터가 중앙 상단에서 제목을 가리고 있으며, 사용자는 우측에서 팔을 뻗어 통계를 안내하는 새 아트와 말풍선 문구 변경을 요청했다.
- 건강지표 카드는 최근 단일 통합 그래프 구조로 바뀌었으나, 사용자는 이전처럼 `체중 & 섭취칼로리 추이`와 `월간 칼로리 리포트`가 보이는 구조로 롤백을 요청했다.
- 미란다 퀘스트 모달 아트는 참조 이미지의 화장, 얼굴, 태도, 표정, 낮은 농도의 선글라스와 깔보는 눈매를 더 반영해야 한다.

## Slice 1. 트레이너 통계 모달 안내 아트

1. `imagegen`으로 통계 모달 전용 트레이너 PNG를 새로 생성한다.
2. 새 트레이너는 우측에 배치해 팔을 왼쪽으로 뻗어 모달 내용을 안내하는 자세로 만든다.
3. 상의는 노골적으로 몸에 달라붙는 검은색 반팔 컴프레션으로 묘사하되, 기존 픽셀/게임 NPC 화풍을 유지한다.
4. 말풍선 문구는 `회원님의 운동 성과를 함께 살펴보시죠!`로 바꾼다.
5. 공유/복사 아이콘은 우측 끝이 아니라 `내 운동 통계` 제목 근처에 붙인다.

## Slice 2. 건강지표 카드 롤백

1. 전체통계와 트레이너 통계 모달의 건강지표 영역을 `체중 & 섭취칼로리 추이` 카드와 `월간 칼로리 리포트` 카드로 바꾼다.
2. `체중 & 섭취칼로리 추이`는 현재 상단 기간 토글의 선택 범위에 맞춰 집계한다.
3. 월간 칼로리 리포트는 현재 월 기준 섭취칼로리, 운동칼로리, 목표, 요약 그리드를 다시 렌더링한다.
4. 기존 `건강 지표` 단일 통합 그래프 UI는 사용자 화면에서 제거한다.

## Slice 3. 미란다 모달 아트 개선

1. 기존 홈 배치용 미란다 스프라이트는 유지하고, 퀘스트 모달 전용 `miranda-npc-seated.png`를 새 버전으로 교체한다.
2. 참조 이미지의 흰/은색 헤어, 차가운 표정, 또렷한 눈화장, 엄격한 태도, 살짝 내려 보거나 깔보는 눈매를 반영한다.
3. 선글라스는 완전 불투명이 아니라 눈매가 보이는 낮은 농도의 틴트 렌즈로 만든다.
4. 기존 모달 좌표와 크기 계약은 유지해 레이아웃 회귀를 줄인다.

## 검증

1. PNG는 RGBA 투명 배경과 실제 파일 헤더를 확인한다.
2. `node --check`로 수정 JS를 확인한다.
3. 관련 테스트와 전체 테스트를 실행한다.
4. `sw.js` `STATIC_ASSETS` 변경 시 `CACHE_VERSION`을 함께 갱신한다.
5. Dashboard3 Pages로 배포하고 `verify:deploy`를 통과시킨다.
